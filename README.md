# ☕ Cafe Management System

A full-stack web application for managing cafe orders, products, and users.

## 🚀 Features
- User Registration & Login
- Product Management
- Shopping Cart
- Order Placement
- Order History

## 🛠️ Tech Stack
- **Backend:** Spring Boot, Spring Security, JWT, JPA/Hibernate
- **Database:** H2 Database
- **Frontend:** HTML, CSS, JavaScript

## 📦 Setup Instructions

### Backend
```bash
cd Backend/backend
./mvnw spring-boot:run
